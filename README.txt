Локальная версия свадебного сайта.

Как открыть:
1. Откройте файл index.html в браузере.
2. Если браузер ограничит локальные скрипты, запустите из этой папки:
   python -m http.server 8123
   затем откройте http://127.0.0.1:8123/

Что работает:
- адаптивная главная страница без фотографии;
- фоновая музыка по кнопке;
- таймер до даты мероприятия;
- блок с основной информацией;
- кнопка маршрута через 2GIS;
- копирование адреса;
- RSVP-форма;
- сохранение ответов в IndexedDB браузера с fallback в localStorage;
- админ-панель по адресу /admin;
- экспорт ответов RSVP в CSV.

Для Vercel:
- vercel.json перенаправляет /admin на index.html;
- после деплоя админ-панель будет доступна по https://ваш-домен/admin.

Примечание:
Ответы RSVP хранятся в IndexedDB браузера (с fallback в localStorage) и доступны
админу только на том же устройстве, где их отправляли. Для общей онлайн-базы
потребуется backend/API или хостинг с базой данных.
